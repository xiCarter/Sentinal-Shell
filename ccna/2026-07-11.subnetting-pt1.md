### Key Concepts
- Subnetting divides a large network into smaller more manageable segments
- CIDR notation: /24 = 256 addresses, /25 = 128, /26 = 64
- Network address = first address in subnet — not assignable
- Broadcast address = last address in subnet — not assignable
- Usable hosts = 2^(host bits) - 2
- Borrowing host bits = more subnets, fewer hosts per subnet
