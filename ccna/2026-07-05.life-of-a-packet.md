### Completed

### Key Concepts
- IP addresses stay constant end to end — never change
- MAC addresses change at every hop — each router rewrites src/dst MACs
- Router de-encapsulates the frame, reads the IP, re-encapsulates with new MACs, forwards
- ARP resolves the next-hop MAC at each router before forwarding — per-link
- If ARP fails at any hop the packet dies there
- Simulation mode shows encapsulation and de-encapsulation at each device in real time
