### Key Concepts
- Full duplex = transmit and receive simultaneously — no collisions
- Half duplex = one direction at a time — collision detection required (CSMA/CD)
- Full and half duplex cannot coexist on a link without causing collisions
- Hubs = half duplex only, broadcast to every port, legacy technology
- Switches = full duplex, smart forwarding by MAC address
- F0/x interfaces = Fast Ethernet, max 10/100 Mbps
- G0/x interfaces = Gigabit Ethernet, 1000 Mbps and above
- interface range = configure multiple ports in one command instead of one by one
- Unused interfaces should be shut down and described as not in use — security best practice
