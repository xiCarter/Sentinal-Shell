### Completed

- Converted values between binary and hexadecimal

### Key Concepts
- IPv4 addresses are 32 bits divided into 4 octets
- Class A: 1-126, Class B: 128-191, Class C: 192-223
- Binary conversion: each octet is 8 bits, values 128/64/32/16/8/4/2/1
- Hexadecimal is base-16 — 0-9 then A-F, used heavily in MAC addresses
  and IPv6
