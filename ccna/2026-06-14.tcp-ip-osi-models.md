### Concepts Locked In
- OSI Model has 7 layers: Physical, Data Link, Network,
  Transport, Session, Presentation, Application
  (mnemonic: Please Do Not Throw Sausage Pizza Away)
- TCP/IP Model has 4 layers: Network Access, Internet,
  Transport, Application
- OSI is the conceptual model — TCP/IP is what's
  actually implemented in real networks
- Each layer has a specific job and communicates only
  with the layers directly above and below it
- PDUs (Protocol Data Units) change name per layer:
  Data → Segment → Packet → Frame → Bits
- Encapsulation: data gets wrapped with headers as it
  moves down the stack
- De-encapsulation: headers are stripped as data moves
  up the stack on the receiving end
