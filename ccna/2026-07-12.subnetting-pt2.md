### Key Concepts
- /25 = 128 addresses, 126 usable, 2 subnets from a /24
- /26 = 64 addresses, 62 usable, 4 subnets from a /24
- /27 = 32 addresses, 30 usable, 8 subnets from a /24
- /28 = 16 addresses, 14 usable, 16 subnets from a /24
- To find the network address: zero out the host bits
- To find the broadcast: turn all host bits to 1
- Subnet increment = 256 - subnet mask octet value
  e.g. /26 = mask 255.255.255.192, increment = 256 - 192 = 64
  So subnets start at .0, .64, .128, .192
- First usable host = network address + 1
- Last usable host = broadcast address - 1
