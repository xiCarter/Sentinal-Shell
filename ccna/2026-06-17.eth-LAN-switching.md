### Concepts Locked In
- Ethernet frame structure:
  Preamble (7 bytes) — allows devices to sync
  before data arrives
  SFD / Start Frame Delimiter (1 byte) — signals
  the actual frame is beginning, acts as delimiter
  Destination MAC (6 bytes)
  Source MAC (6 bytes)
  Type/Length (2 bytes) — identifies Layer 3
  protocol (IPv4, IPv6 etc)
- Hexadecimal: base-16 numbering system
  digits 0-9 then A-F (16 total values)
  MAC addresses are written in hex
- How switches learn MAC addresses:
  Unknown destination → flood all ports
  Source MAC on incoming frame → recorded in
  MAC address table dynamically
  Known destination → forward only to that port
- MAC address table built dynamically through
  normal traffic — no manual configuration needed
