### Key Concepts
- Max hosts = 2^(host bits) - 2 — subtract network and broadcast address
- Router interfaces are administratively down by default — must run no shutdown
- show ip interface brief shows all interfaces, IPs, and up/down status
- ARP operates per-network — a router rejects an ARP frame arriving on an
  interface if the source IP is outside that interface's subnet
- Simulation mode in Packet Tracer shows OSI layer messaging per hop —
  used this to isolate where the ARP was being rejected
- A single mistyped subnet can silently break an entire network segment
  while leaving others functional — narrow down by pinging hop by hop
